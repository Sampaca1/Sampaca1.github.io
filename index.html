Note: new download instructions

  1. If you don't have the executable (You only need one, even across versions):
    1. Click the download for the executable
    2. Go into the download pane
    3. Click where it says unverified download blocked
    4. Click download unverified file
  2. Download the latest pck
  3. Make sure the pck and the executable are in the same folder (that folder can just be downloads if you want)
  4. Double click "Out at dark.exe" to play
