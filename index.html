Download Latest Version

  1. Download zip file via link of choice
  2. Go to zipped file in downloads
  3. Click on zip once
  4. Click extract all at the top
  5. Click extract
  6. Open folders
  7. Double click the file called "Out at dark.exe"
  8. If a Windows Protection pop-up appears, click more info and then run anyway
  9. Play the game! (Game instructions included in-game)
